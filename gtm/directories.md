# Directory & resource-list submissions

Free backlinks + drip traffic. Submit over 2–3 weeks (a natural link velocity
looks better than 20 in one day). Reuse this copy everywhere:

**One-liner:** Free, statute-cited legal calculators and document generators
for small landlords — deposit interest, late-fee caps, rent-increase notices,
deposit deadlines, receipts. No signup.

**Longer blurb:** LandlordKit gives independent landlords free state-aware
tools: security deposit interest calculators, late-fee cap checkers, rent
increase notice generators, deposit-return trackers, rent receipts, and more.
Every state rule cites the statute with a last-verified date. Tools run in the
browser and export clean PDFs — no account required. Pro ($12/mo) adds saved
properties, batch documents, and custom branding.

- **Product Hunt** (see product-hunt.md — do this one deliberately)

- **Uneed**, **PeerLists**, **BetaList** (if pre-PH), **MicroLaunch**
- **IndieHackers** product page + a build-log post (the "solo founder, $20/mo
  stack, 204 programmatic pages" story performs well there)

## Landlord / REI resource lists (higher-value links)
- **BiggerPockets** — FilePlace/resources + an introduction post in the tools thread
- **r/Landlord wiki** — message mods asking if the free calculators fit the
  sidebar/wiki resources (they sometimes add statute-cited tools)
- **AAOA (American Apartment Owners Association)** — resources page suggestion
- **Local/state landlord association resource pages** — start with the states
  you're verified in (MA, NY, CO, ME, TX); email their admin with the matching
  state page link, not the homepage
- **Law-school / legal-aid "tenant-landlord resources" pages** — they link
  calculators that cite statutes; offer the deposit-return tracker
- **Nolo / Avvo community answers** — answer questions, cite the state page
